# Типы контента

- application/json
- application/xml
- text/plain
- text/html
- application/octet-stream
- image/jpeg
- image/png
- multipart/form-data
- application/x-www-form-urlencoded
- application/[Any]
- image/[Any]

> **[Any]** - любое значение, которого нет в списке
