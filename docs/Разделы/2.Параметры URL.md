# Параметры URL (path)

Формат записи:
```bsl
//  Параметры URL:
//      * [ИмяПараметра] - [ТипПараметра] - [ОписаниеПараметра]
```
> **[ИмяПараметра]** - имя параметра (без спец.символов)

> **[ТипПараметра]** - [тип](../Типы/README.md) параметра

> **[ОписаниеПараметра]** - необязательное многострочное описание параметра
